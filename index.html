Sublime Video

A lightweight, CSS-skinnable custom HTML5 video player.

Download JS Download CSS Docs Options

Demo

Tip: Click video to play/pause. Hotkeys: K/Space play/pause, ←/→ skip, M mute, F fullscreen.

Why Sublime Video?

  • Drop-in: add a class to your <video>
  • Skinnable: theme everything via CSS variables
  • Responsive: flex layout, sane mobile fallbacks
  • Accessible: focusable wrapper + aria-labels

Getting started

<link rel="stylesheet" href="/path/dist/sublime-video.css">
<script defer src="/path/dist/sublime-video.js"></script>

<video class="sblm-video"
  src="/media/video.mp4"
  poster="/media/poster.jpg"
  preload="metadata"
  data-theme="emerald"
  data-accent="#22c55e"
  data-skip="10">
</video>

Programmatic API

// Default manager:
const mgr = window.SublimeVideo.manager;

const video = document.querySelector('video.sblm-video');
const player = mgr.get(video);

player.play();
player.pause();
player.seekBy(10);
player.setAccent('#ef4444');

Options

Configure Sublime Video via data-* attributes on the <video>. Themes are CSS variables (preset or custom).

Core data-attributes

AttributeWhat it doesExample
data-accent Sets the player accent color (--sv-accent). data-accent="#22c55e"
data-theme Applies a preset theme class (ex: .sv-theme-emerald). data-theme="emerald"
data-autohide Auto-hides controls while playing when idle. data-autohide="true"
data-hotkeys Enables keyboard shortcuts on the player (when focused). data-hotkeys="true"
data-skip Default skip step for skip buttons (seconds). User can change in Settings. data-skip="10"
data-pip Shows Picture-in-Picture button if supported. data-pip="true"

Settings popover

  • Skip Buttons: 10s / 30s / 60s / 90s
  • Playback Speed: 0.75x / 1x / 1.25x / 1.5x / 2x

Settings persist using localStorage (per browser).

Theming

Themes are just CSS variables — use presets or create your own: 

/* Example: custom theme */
.my-sv-theme {
  --sv-accent: #ef4444;
  --sv-bg: rgba(0,0,0,0.62);
  --sv-radius: 18px;
}
